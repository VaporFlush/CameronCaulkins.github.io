Conway's Game of Life

Game of Life made with WXwiggets

Conway's Game of Life

   In a month-long endeavor, I embarked on recreating Conway's Game of Life using WXWidgets, showcasing my prowess in game development and GUI design. Using WXWidgets, I made a graphical interface, allowing users to engage with the simulation. Central to the project was the implementation of Conway's rules, dynamically updating cell states based on neighboring conditions, a testament to my algorithmic proficiency. The project boasts features such as customizable grid sizes, adjustable simulation speeds, and pattern saving/loading, enhancing its functionality and user appeal. With attention to detail, debugging, and effective time management, I successfully realized this project, underscoring my capabilities as a game developer and GUI designer.
